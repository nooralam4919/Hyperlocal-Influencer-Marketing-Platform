# Hyperlocal Influencer Marketing Platform

A full-stack platform that connects local businesses with nearby influencers to create targeted marketing campaigns. The application streamlines influencer discovery, collaboration, and campaign management through an intuitive and responsive user experience.

## Features

* Influencer discovery and profile management
* Business and creator authentication
* Campaign creation and management
* Secure user authorization using JWT
* Media upload and management
* Responsive and modern user interface
* RESTful API integration
* Real-time collaboration workflow

## Tech Stack

### Frontend

* React.js
* JavaScript
* Tailwind CSS
* React Router
* Axios

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Cloudinary

## Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Hyperlocal-Influencer-Marketing-Platform.git
cd Hyperlocal-Influencer-Marketing-Platform
```

### Frontend Setup

```bash
cd influencer-platform-frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd influencer-platform-backend
npm install
npm start
```

## Environment Variables

Create a `.env` file in the backend directory and configure:

```env
PORT=
MONGODB_URI=
JWT_SECRET=
CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
```
